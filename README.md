# 🎨 Design Prompts Collection

웹 디자인 프롬프트 모음집. AI 코딩 에이전트에게 디자인 시스템을 전달할 때 사용.

## 사용법
각 `.md` 파일을 AI 코딩 도구(Cursor, Claude Code, Codex 등)에 프롬프트로 전달하면 해당 디자인 스타일로 웹사이트를 생성합니다.
