## 🚀 Usage

각 프롬프트 파일에는:
1. **레퍼런스 이미지** — 스타일 결과물 시각화
2. **Summary** — 핵심 특징 요약
3. **Style Prompt** — AI에 직접 입력 가능한 스타일 정의
4. **Layout & Structure** — 각 섹션별 구현 지침
5. **Special UI Components** — 독특한 컴포넌트 구현법

Claude, Cursor, v0, Lovable, Bolt 등 AI 코딩 도구에 프롬프트를 붙여넣으면 해당 스타일로 UI를 생성합니다.
